# 김주인내과 웹사이트

부산 서면 김주인내과 공식 웹사이트입니다.
kimjooin.com에 GitHub Pages로 배포 중입니다.

## 포함 항목
- `index.html` — 메인 홈페이지
- `Module/` — 진료안내, 오시는길 등 서브페이지
- `en.html`, `cn.html`, `jp.html` — 영문/중문/일문 안내 페이지
- `assets/css/clinic-revamp.css` — 전체 스타일
- `assets/js/clinic-revamp.js` — 인터랙션 스크립트
- `assets/fonts/JejuMyeongjo.otf` — 제주명조 폰트
- `images/custom/` — 병원 시설 사진
- `robots.txt`, `sitemap.xml`, `CNAME`
